## 5. 结论

### 核心发现

1. **本体注入显著提升 LLM 准确率**：ontology_llm 较 bare_llm 准确率提升 **+35.9%**，说明结构化本体为模型提供了可靠的推理锚点。

2. **本体降低幻觉率**：有本体约束时，LLM 倾向于引用具体公理/关系/规则，幻觉率从 0.712 降至 0.446。

3. **符号推理无需 LLM 调用**：延迟接近 0ms，结论附带完整证明链，适合作为高可靠性基线。但其对自然语言语义解析能力有限。

4. **反事实推理是符号推理的独特优势**：counterfactual 类别中，符号推理可以精确计算假设变更的级联影响，而 LLM 只能给出模糊推测。

### 适用场景建议

| 场景 | 推荐条件 | 理由 |
|------|----------|------|
| 需要可验证的确定性推理 | symbolic | 零幻觉，有证明链 |
| 需要自然语言交互 | ontology_llm | 语义理解强+本体约束 |
| 快速原型/无本体可用 | bare_llm | 零额外开发成本 |
| 反事实/影响分析 | symbolic | 可精确计算假设变更 |
| 生产环境高可靠推理 | symbolic + ontology_llm | 符号验证+自然语言 |
