# 电商系统产品介绍

## 客户体系

我们的平台服务于两类客户：个人客户和企业客户。个人客户是指以自然人身份注册的用户，企业客户是指以法人身份注册的用户。

VIP客户是一种特殊的客户类型，指年消费额超过10万元的客户群体。VIP客户享有专属客服和折扣优惠。

客户可以通过手机号或邮箱注册账户。注册后，客户可以浏览商品、下单购买、查看物流状态。

## 商品分类

商品分为多个类别：电子产品、服装、食品等。每个类别下有多个子类别。

智能手机是一种电子产品，属于数码产品的大类。iPhone是我们热销的智能手机品牌之一。

服装分为男装、女装、童装三个子类别。每个子类别下又分为上装、下装、配饰等。

## 订单流程

客户下单后，订单包含一个或多个商品。每个订单项记录了商品和购买数量。

订单支付后进入发货流程。订单状态包括：待支付、已支付、已发货、已完成、已取消。

如果客户在24小时内未支付，订单将自动取消。

## 物流配送

订单完成后由物流公司配送。每个物流公司有自己的配送范围和时效标准。

物流状态包括：已揽收、运输中、已到达、已签收。客户可以通过订单号查询物流状态。

## 支付方式

我们支持多种支付方式：微信支付、支付宝、银行卡支付。企业客户还可以使用对公转账。

支付完成后，系统会自动更新订单状态为已支付，并通知商家发货。

## 权限管理

管理员可以查看和修改所有订单信息。客服人员可以查看订单详情，但不能修改价格。

普通客户只能查看自己的订单。VIP客户有权查看专属商品目录和历史消费记录。

企业客户不能删除已完成的订单记录。

## 数据规范

客户手机号必须为11位数字，格式为1开头。邮箱地址必须包含@符号。

订单金额最小值为0.01元，最大值为999999.99元。商品名称长度不超过200个字符。

收货地址最长不超过500个字符。

## 业务操作流程

完整的下单流程如下：浏览商品 → 加入购物车 → 确认订单 → 选择支付方式 → 完成支付。

支付成功后，系统会自动更新库存数量，并通知仓库进行拣货。

退款流程：客户提交退款申请 → 客服审核 → 财务退款 → 退款到账。

如果客户在48小时内未确认收货，系统将自动确认收货。
